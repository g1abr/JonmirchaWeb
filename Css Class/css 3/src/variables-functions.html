Funciones en CSS

  • Var() Recupera valores de las Custom Properties
  • Url() Permite especificar una URL de una imagen o recurso externo.
  • min() & max() Retornan el valor mínimo o máximo entre varias opciones, respectivamente. Útil para establecer tamaños o márgenes que se ajusten de manera óptima en diferentes dispositivos.
  • Calc() Realiza cálculos con diferentes unidades, combinando píxeles, porcentajes, em, entre otros. Es útil para lograr layouts más flexibles
  • clamp() Define un valor que se adapta automáticamente entre un límite mínimo, un valor preferido y un límite máximo.
