Pixels (px)

Un píxel o pixel, ​ plural píxeles o pixeles es, en informática, la unidad mínima que forma una imagen digital. Esta imagen puede ser mostrada e interpretada por el hardware de un ordenador, por ejemplo, mediante un monitor o una impresora.

Ems

Son unidades relativas, así que más exactamente 1 em equivale al cien por cien del tamaño inicial de fuente. Si el usuario ha configurado su navegador para que el tamaño de fuente inicial sea 20 píxeles, entonces éste será el tamaño de 1 em.
